# Kotlin_Practices

This repository contains practice code for learning and improving Kotlin programming fundamentals.  
It covers essential concepts such as variables, data types, operators, control flow, loops, functions, and object‑oriented programming.

---

## 📂 Project Structure

- **Variable.kt** → Basics of variable declaration and usage  
- **DataType.kt** → Working with different data types  
- **Operators.kt** → Arithmetic, relational, and logical operators  
- **ifElseCondition.kt** → Conditional statements  
- **WhenStatement.kt** → Kotlin’s `when` expression  
- **ForLoop.kt / WhileLoop.kt / DoWhileLoop.kt** → Looping constructs  
- **array_1.kt / array_cantinMenu.kt** → Array handling and menu example  
- **Functions.kt / ReturnTypeFunction.kt / OverloadingFunction.kt** → Function definitions, return types, and overloading  
- **OOP/** → Object‑Oriented Programming concepts (classes, objects, inheritance, etc.)  
- **Main.kt** → Entry point for running examples  

---

## 🎯 Learning Outcomes

By exploring these files, you will:

- Understand Kotlin syntax and core concepts  
- Practice control flow and looping mechanisms  
- Learn how to define and use functions effectively  
- Gain hands‑on experience with arrays and collections  
- Strengthen knowledge of object‑oriented programming in Kotlin
